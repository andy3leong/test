## Development Stacks

- Sqlite for Database
- Ruby On Rails for API backend
- frontend Boilerplate with create-react-app
- Bootstrap CSS


## Installation

### backend

```
rvm install 2.4.1
rvm use 2.4.1

bundle install
bundle exec rake db:migrate

rails s
```

### FrontEnd

```
NVM
nvm install 10.11.0
nvm use 10.11.0

npm install && npm start
```

Open any browser and visit http://localhost:3000
